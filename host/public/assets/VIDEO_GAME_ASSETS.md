6. **Place**: In `host/public/assets/models/dice/`

## File Conversion

If you get DAE, BLEND, or OBJ files:

1. **Download Blender**: https://www.blender.org/download/
2. **Import**: File ‚Üí Import ‚Üí [Format]
3. **Export**: File ‚Üí Export ‚Üí glTF 2.0
4. **Choose**: "glTF Binary (.glb)"
5. **Save**: To appropriate directory
