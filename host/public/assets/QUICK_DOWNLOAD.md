# Quick Asset Download Guide

## Essential Assets (Required for Best Visuals)

### 1. Character Models
**Source**: https://kenney.nl/assets/blocky-characters
- Click "Download" button
- Extract the ZIP
- Find 4 character GLB files
- Rename them to: `character_1.glb`, `character_2.glb`, `character_3.glb`, `character_4.glb`
- Place in: `host/public/assets/models/characters/`

### 2. Pipe Model
**Source**: https://poly.pizza/m/f1A1MuUQfC3
- Click the "Download" button on the page
- Select "GLB" format
- Save as: `host/public/assets/models/pipe.glb`

### 3. Coin Model
**Source**: https://poly.pizza/m/7IrL01B97W
- Click "Download" → Select "GLB"
- Save as: `host/public/assets/models/coin.glb`

### 4. Star Coin Model
**Source**: https://poly.pizza/m/n5nJIQAozN
- Click "Download" → Select "GLB"
- Save as: `host/public/assets/models/star_coin.glb`

### 5. Tree & Rock Models
**Source**: https://poly.pizza/bundle/Ultimate-Stylized-Nature-Pack-zyIyYd9yGr
- Click "Download Bundle"
- Extract the ZIP
- Find `tree.glb` and `rock.glb`
- Place in: `host/public/assets/models/nature_pack/`

## Optional Assets (Have Procedural Fallbacks)

### Shop Model
- Search poly.pizza for "shop" or "store"
- Download as GLB
- Save as: `host/public/assets/models/shop.glb`

### Game Reference Props
All have procedural fallbacks, but you can add custom models:
- Doom imp: `models/props/doom_imp.glb`
- Pacman ghost: `models/props/pacman_ghost.glb`
- Tetris block: `models/props/tetris_block.glb`
- Space invader: `models/props/space_invader.glb`
- Arcade cabinet: `models/props/arcade_cabinet.glb`
