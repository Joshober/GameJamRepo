## Asset Directories

Each asset subdirectory contains a README with specific instructions:
- `Assets/platforms/` - Platform textures
- `Assets/blocks/` - Broken block textures
- `Assets/props/` - Props (bridges, tubes, flags)
- `Assets/backgrounds/` - Background textures
- `Assets/portals/` - Portal effect sprites

## Files

- `main.py` - Main game entry point
- `manifest.json` - Minigame manifest for the board game system
- `mobile_controls.py` - Mobile control support
- `generate_assets.py` - Asset generation script
