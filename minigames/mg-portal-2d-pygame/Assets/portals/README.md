# Portal Effect Sprites

Place portal effect sprites here.

## Recommended Assets

**2D Pixel Art Portal Sprites** (https://elthen.itch.io/2d-pixel-art-portal-sprites)
- Portal sprite sheets with animations
- Idle, Emerge, Disappear animations
- Green and purple variants

## File Naming Convention

- `portal_blue_idle.png` - Blue portal idle animation frame
- `portal_blue_emerge.png` - Blue portal emerge animation frame
- `portal_blue_disappear.png` - Blue portal disappear animation frame
- `portal_orange_idle.png` - Orange portal idle animation frame
- `portal_orange_emerge.png` - Orange portal emerge animation frame
- `portal_orange_disappear.png` - Orange portal disappear animation frame

Or use sprite sheets:
- `portal_blue_sheet.png` - Blue portal sprite sheet
- `portal_orange_sheet.png` - Orange portal sprite sheet

## Current Implementation

The game currently uses simple colored circles for portals. Adding sprite-based portals will enhance the visual appeal.
