# Platform Textures

Place platform texture tiles here. Recommended assets:

1. **Free Puzzle Platformer Kit** (https://paweljarosz.itch.io/puzzle-platformer-asset-mnimalistic-game-kit-portal-like)
   - Extract grey platform tiles from the kit
   - Recommended files: platform tiles, floor tiles, wall tiles

2. **2D Platformer Pack** (https://chequered.ink/2d-platformer-pack/)
   - Extract tiles from industrial/sci-fi tilesets
   - Look for grey concrete or metal platform textures

## File Naming Convention

- `platform_grey.png` - Main grey platform texture (will be tiled)
- `platform_edge.png` - Platform edge texture (optional)
- `wall_grey.png` - Grey wall texture (optional)

The code will automatically tile these textures across platform surfaces.
