# Background Elements

Place background tiles and patterns here.

## Recommended Assets

1. **2D Platformer Pack** (https://chequered.ink/2d-platformer-pack/)
   - Includes 24 backgrounds
   - Extract backgrounds matching muted beige color scheme
   - Look for abandoned/industrial backgrounds

2. **Custom Backgrounds**
   - Muted beige color (#F5F5DC or similar)
   - Faint room outline patterns
   - Optional: vine/overgrowth patterns

## File Naming Convention

- `background_main.png` - Main background texture (will be tiled)
- `background_overlay.png` - Overlay pattern (optional)
- `room_outline.png` - Faint room outline pattern (optional)

## Usage

Backgrounds can be:
- Tiled across the entire screen
- Used as parallax layers
- Applied as overlay patterns

If no background files are found, the code uses the background color from level design.
